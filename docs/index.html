What it does

Built for power users.

Every feature you need for Instagram analysis, designed to be fast, private, and effortless.

HD Profile Pictures

Download any public Instagram profile picture in full 1080×1080 resolution with a single click.

Unfollower Detection

See exactly who stopped following you back and unfollow them in bulk — with smart rate limiting for account safety.

Follower Analytics

Track gains and losses over time with interactive charts. Spot trends before they happen.

Data Export

Save your profile data as JSON, CSV or TXT for offline analysis or record keeping.

Full Customization

5 color themes, 9 font families, and dark/light mode. Make it yours.

Privacy First

Zero external servers. All processing happens locally in your browser. Your data stays yours.

Manual Install

Takes about 2 minutes

1
Download the ZIP

Get the latest release from GitHub.

Download ZIP
2
Extract the ZIP

Unzip to any folder on your computer.

3
Open Extensions Page

Chrome: chrome://extensions
Edge: edge://extensions
Opera: opera://extensions

4
Enable Developer Mode

Toggle the "Developer mode" switch (top-right corner).

5
Load Unpacked

Click "Load unpacked" and select the extracted folder.

6
Done! 🎉

Open Instagram and click the WeXProfile icon to get started.
